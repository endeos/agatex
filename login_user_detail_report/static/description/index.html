Community Enterprise

Features

  • This module tracking user login details. like,
    • Login date
    • IP Address
    • Host(Server) Name
  • We can also print the report for selected login users.

Configuration

  • For configuration only have to install the login user detail report module.

Print the report for specific time

  • You can enter the Start and End date for print the report of login user between that time.

Print the report for all user

  • You have to check All Users for print the Report of Every User.

Print Report For Selected User

  • You have to check Selected User for print the Report of Some user only.
  • If User has no login record then this message will appear.
